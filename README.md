# Mozhimobility
# Bus Scheduling Website with Environmental Awareness

---

## Description
This project is a multilingual bus scheduling website designed to facilitate users in managing their bus routes, schedules, and environmental impact. Built using Python with Django framework, SQLite3 database, HTML, CSS, JavaScript, and various libraries including Pillow and Bootstrap, the website aims to provide a user-friendly interface for bus operators and passengers while promoting environmental awareness through eco-friendly practices.

## Features
- **Multilingual Support**: The website supports multiple languages to cater to diverse user demographics.
- **Bus Scheduling**: Users can manage bus routes, schedules, and timings efficiently through an intuitive interface.
- **Environmental Awareness**: The website emphasizes environmental consciousness by providing information about eco-friendly practices in transportation and promoting sustainable commuting options.
- **Responsive Design**: The website is designed with responsiveness in mind, ensuring optimal user experience across various devices and screen sizes.
- **Customizable Styles**: Built using HTML, CSS, and JavaScript, the website offers customizable styles and layouts to suit user preferences.
- **Integration with Pillow Library**: The project integrates the Pillow library to handle image processing tasks, such as image uploads and manipulations.
- **Bootstrap Framework**: Utilizing the Bootstrap framework, the website features sleek and modern design elements, enhancing user interaction and visual appeal.

## Requirements
- Python 3.x
- Django 3.x
- SQLite3
- Pillow Library
- Bootstrap Framework

## Installation
1. Clone the repository to your local machine.
2. Install Python and Django if not already installed.
3. Navigate to the project directory and set up a virtual environment.
4. Install project dependencies using pip.
5. Run migrations to create SQLite3 database tables.
6. Start the Django development server.

## Usage
1. Access the website through a web browser.
2. Register an account or log in as a guest.
3. Explore bus routes, schedules, and environmental awareness resources.
4. Customize settings and preferences as desired.
5. Contribute to environmental sustainability by adopting eco-friendly transportation practices.
